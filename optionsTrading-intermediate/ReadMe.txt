The downloadable resources folder contains the Ipython notebooks.

---------------------------------------------------------------------------------------------------------------------------------------------------------------------

The Section 1 folder contains the following file.

Theoretical price of option -- An Ipython notebook to calculate the call and put prices using the mibian package.


---------------------------------------------------------------------------------------------------------------------------------------------------------------------

The Section 3 folder contains the following files.

Greeks calculator -- An Ipython notebook to to calculate the Greeks (Delta, Gamma, Vega, Theta, and Rho) in Python using the mibian library.

Option price using Delta and Gamma -- An Ipython notebook to calculate the approximate price of call option using Delta and Gamma.

Option ptrice using Vega -- An Ipython notebook to calculate the approximate price of call option using Vega.

---------------------------------------------------------------------------------------------------------------------------------------------------------------------

The Section 4 folder contains the following file.

Calendar Spread -- An IPython notebook to create payoff graph for Calendar Spread.


---------------------------------------------------------------------------------------------------------------------------------------------------------------------

The Section 5 folder contains the following files.

Forward Volatility -- An Ipython notebook to create a forward volatility strategy on Nifty call option.

Nifty.csv - The options data file for the Forward Volatility code

Volatility Smile -- An Ipython notebook to create a trading strategy using volatility smile on the Nifty call options.

Option_data_NIFTY.csv - The options data file for the Forward Volatility code
